# 📸 Adding the Voting Image

## Setup Instructions

To display the online-voting.png image on the home page, follow these steps:

### Step 1: Copy the Image
1. Locate your image at: `D:\6th Semester\online-voting.png`
2. Copy this image file

### Step 2: Paste into Public Folder
Paste the image into:
```
frontend/public/online-voting.png
```

### Step 3: Refresh Browser
- Refresh your browser at `http://localhost:3000`
- The image will now appear on the left side of the home page

## 🎨 Image Features

The image will display with:
- ✨ Beautiful fade-in animation from left
- 🎯 Floating animation (gentle up and down motion)
- 💫 White glow effect around the image
- 📱 Responsive design (adapts to mobile screens)
- 🔄 Positioned on the left side of hero section
- 📊 Next to the "Secure • Transparent • Digital Voting" title

## 📝 File Structure

```
frontend/
├── public/
│   ├── index.html
│   └── online-voting.png    ← Place your image here
└── src/
```

## 🖼️ Image Requirements

- **Recommended size**: 400-600px wide
- **Format**: PNG, JPG, or WebP
- **Aspect ratio**: Any ratio works (will scale responsively)

## ✅ Done!

Once the image is in `frontend/public/online-voting.png`, it will automatically display on the home page with animations.
